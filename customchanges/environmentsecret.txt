✅ Read (resourceGithubActionsEnvironmentSecretRead)
	•	Checks if repo exists → removes state if not.
	•	Checks if repo is archived → removes state if so.
	•	Checks if environment exists → removes state if not.
	•	Checks if secret exists → removes state if not.
	•	Sets timestamps correctly.

This is exactly what Terraform expects, so terraform plan/apply will handle deleted repos/environments/secrets correctly.

⸻

✅ Update (resourceGithubActionsEnvironmentSecretUpdate)
	•	Handles 404 on CreateOrUpdateEnvSecret → removes state if repo/environment/secret deleted.
	•	Updates timestamps from GitHub.
	•	Encrypts plaintext if needed.

Perfect for safely handling updates, including edge cases.

⸻

✅ Delete (resourceGithubActionsEnvironmentSecretDelete)
	•	Handles 404 → removes state if already deleted.
	•	Logs info.
	•	Idempotent, safe for Terraform. ✅
